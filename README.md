# BotoEducationTest
Test task for Boto Education

# Запуск:
 - Для корректного запуска необходимо предаврительно установить uv и uvicorn
 - Склонируйте репозиторий
 - Выполните в корне две команды:
 - `uv sync`
 - `uvicorn main:app`
 - Приложение запущено и доступно по адресу http://127.0.0.1:8000

# Документация:
http://127.0.0.1:8000/docs

# Тесты:
- Если у вас установлен pytest, вы можете легко запустить все тесты, выполнив в корне команду:
- `pytest`

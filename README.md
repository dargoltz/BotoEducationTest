# BotoEducationTest
Test task for Boto Education

# Запуск:
 - Для корректного запуска необходимо предаврительно установить uv
 - Склонируйте репозиторий
 - Выполните в корне следующие команды:
 - `uv sync`
 - `source .venv/bin/activate`    
 - `uvicorn main:app`
 - Приложение запущено и доступно по адресу http://127.0.0.1:8000

# Документация:
http://127.0.0.1:8000/docs

# Тесты:
- Вы можете легко запустить все тесты, выполнив в корне команду:
- `pytest`
